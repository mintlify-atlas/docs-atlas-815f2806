## How HASH works

<Steps>
  <Step title="Data enters the graph">
    Entities are created from web pages, integrations, API calls, or directly through the HASH UI. Every entity has a type that defines its schema.
  </Step>
  <Step title="AI agents enrich and validate">
    Autonomous AI workers run flows to extract, link, and check entities — keeping your graph accurate and up-to-date without manual effort.
  </Step>
  <Step title="Query and use your data">
    Retrieve entities using structural queries or HashQL, connect to external tools via the Graph API, or browse visually in the HASH workspace.
  </Step>
</Steps>

<Note>
  HASH is available as a managed cloud service at [app.hash.ai](https://app.hash.ai) or as a self-hosted deployment. The cloud version requires no infrastructure setup and is the fastest way to get started.
</Note>
